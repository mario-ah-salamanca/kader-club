# Build Add Card flow

## Goal

Let users add a card from the catalog into their collection.

## Fields

- Condition
- Quantity
- Purchase price
- Grade
- Trade status

## Acceptance Criteria

- [ ] User can add a card from detail page.
- [ ] Required fields are validated.
- [ ] Add-another behavior is considered.
- [ ] Trade status can be set.
- [ ] Success state confirms the card was added.

## Labels

`type: feature`, `priority: P0`, `area: engineering`, `area: design`
