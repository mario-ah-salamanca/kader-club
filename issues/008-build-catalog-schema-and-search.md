# Build catalog schema and search

## Goal

Allow users to search football-card variants by meaningful collector fields.

## Search Fields

- Player
- Club
- League
- Set
- Season
- Parallel

## Acceptance Criteria

- [ ] Search returns seeded cards.
- [ ] Search supports player, club, set, season, and parallel.
- [ ] Card detail page shows useful metadata.
- [ ] Empty state explains what to do if a card is missing.

## Labels

`type: feature`, `priority: P0`, `area: engineering`, `area: data`
