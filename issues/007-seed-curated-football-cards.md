# Seed 100–300 curated football cards

## Goal

Create a high-quality seed catalog for beta testing.

## Suggested Scope

Start with:

- Topps Chrome Bundesliga
- Topps Chrome UEFA
- Topps Merlin
- Panini Prizm
- Panini Select
- Sapphire
- Finest

## Acceptance Criteria

- [ ] 100–300 cards imported.
- [ ] Required metadata is complete.
- [ ] At least 80% of catalog cards have images.
- [ ] Duplicate handling is defined.
- [ ] Data quality issues are tracked.

`type: task`, `priority: P0`, `area: data`
