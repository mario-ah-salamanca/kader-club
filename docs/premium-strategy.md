# Premium Strategy

## Monetization Principle

Do not monetize before trust is established.

First prove that users will add real collections and return to update them.

## Free Tier

Positioning:

> Start your digital football-card binder.

Features:

- Limited collection tracking
- Basic estimated collection value
- Basic catalog search
- Basic wishlist
- Basic trade matches

Validation signal:

- User adds 10+ cards.
- User returns weekly.

## Collector Plus

Positioning:

> Build, showcase, and trade smarter.

Target users:

- Active collectors
- Hobby traders

Possible features:

- Unlimited collection size
- Public showcase profile
- Wishlist alerts
- Advanced trade matching
- Set completion tracker
- CSV import/export
- Manual trade history
- Multiple future squads
- Squad themes

Validation signal:

- User repeatedly uses wishlist and trade matches.

## Portfolio Pro

Positioning:

> Manage your collection like a portfolio.

Target users:

- Power traders
- Investor collectors

Possible features:

- Advanced ROI analytics
- Historical valuation charts
- Price alerts
- Comps database
- Bulk import/edit/export
- Liquidity score
- Deal scanner
- Market dashboards

Validation signal:

- User imports a large collection and tracks profit/loss.
