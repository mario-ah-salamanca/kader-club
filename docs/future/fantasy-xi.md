# Future Feature: Fantasy XI

## Strategic Purpose

Kader Club should not become only a finance dashboard for football cards.

Fantasy XI preserves the emotional side of collecting:

- Pride
- Nostalgia
- Rarity
- Club identity
- Player attachment
- Joy of building something personal

## Core Concept

> Turn your collection into your club.

Users can build a football squad using only cards they actually own.

## Product Timing

Do not build Fantasy XI in the first MVP.

The MVP must first prove that users will:

- Add real collections
- Trust the catalog
- View valuations
- Create wishlists
- Use trade discovery

## Future User Flow

1. User opens My Squad.
2. User chooses a formation, such as 4-3-3, 4-2-3-1, or 3-5-2.
3. User fills positions using cards they own.
4. User chooses a captain card.
5. User adds bench cards.
6. System calculates light squad attributes.
7. User publishes the squad on their profile.
8. System suggests wishlist or trade targets to improve the squad.

## MVP-Compatible Placeholder

Before full Fantasy XI, build:

### Public Profile Showcase

- Featured card
- Favorite 5 cards
- Favorite club
- Collection theme
- Cards open to trade

## Possible Scoring Model

```txt
Squad Score = Collector Identity + Rarity + Completeness + Chemistry
```

## Roadmap Placement

- MVP: collection, catalog, portfolio, wishlist, trade discovery
- Post-MVP 1: public showcase and favorite cards
- Post-MVP 2: set completion and collecting challenges
- Post-MVP 3: Fantasy XI squad builder
- Post-MVP 4: weekly squads, community voting, and trade suggestions based on squad gaps
